# Aplicacion de Fondo de ahorro familiar
**Descripción:** Esta aplicacion es un fondo de ahorro que permite a los familiares ahorrar una parte de sus ingresos en un fondo y generar intereses a traves de prestamos.

**Esta siendo desarrollada 100% con apoyo de AI (AI-driven development).**

## Arquitectura:
**Patrón:** Domain-Driven Design (DDD) + Clean Architecture.
- Arquitectura Orientada a Eventos.

## Stack:
- **Backend:** Node.js + TypeScript + Express.
- **Frontend:** React (Vite) + Tailwind CSS.
- **DB:** SQL Server 2022 (Dockerizado).
- **Contenerización:** Docker Compose para orquestar API, DB, RabbitMQ y frontend app.

- Inicio de Sesión y seguridad  - Done.
- Administracion de Usuarios - Done.
- Creacion cuentas de ahorro - Done.
